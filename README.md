# Digital Garden

A public collection of research, notes, and explorations by [Mark Recek](https://github.com/mrecek).

## Structure

- **Research/** — investigations, comparisons, and analysis of external topics
- **Posts/** — guides, opinions, short takes, observations, commentary

## About

This is a [digital garden](https://maggieappleton.com/garden-history) — a collection of notes at various stages of development, tended over time rather than published as polished articles. Some pieces are seedlings, others are fully grown.
